# 🔰 **仮想ネットワーク（VNet）とは**

## ■ 概要

**Virtual Network（VNet）** とは

👉 **Azure上の仮想ネットワーク**

---

## 💡 一言でいうと

👉 **「Azure内の社内LAN」**

---

# 🧠 **イメージ（超重要）**

```
VNet
 ├ サブネットA
 │   └ VM
 └ サブネットB
     └ DB
```

---

## 🎯 試験ポイント

- Azureリソース同士の通信基盤
- VMは通常VNetに接続

---

# 🌍 **IPアドレス範囲（Address Space）**

---

# 🔰 概要

VNet全体で使うIP範囲

---

## 💡 例

```
10.0.0.0/16
```

---

# 🎯 試験ポイント

👉 CIDR表記を理解する

---

# 🧠 **CIDRとは？（重要）**

## ■ 例

```
10.0.0.0/16
```

---

## 💡 意味

- `10.0.0.0` → ネットワークアドレス
- `/16` → ネットワーク部

---

## 🎯 試験ポイント

👉 `/24` の意味などが出ることあり

---

# 📦 **サブネット（超重要）**

---

# 🔰 概要

VNetを分割したネットワーク

---

## 💡 例

```
VNet: 10.0.0.0/16

Subnet-Web: 10.0.1.0/24
Subnet-DB: 10.0.2.0/24
```

---

## 🎯 試験ポイント

- VMはサブネットに配置
- セキュリティ分離で重要

---

# 🌐 **NIC（Network Interface Card）**

---

# 🔰 概要

VMのネットワークカード

---

## 💡 ポイント

- VMに必須
- IP設定を持つ

---

# 🎯 試験ポイント

👉 VMはNIC経由で通信

---

# 🔥 **Public IPとPrivate IP（超重要）**

---

# ■ Private IP

内部通信

例：

`10.0.1.4`

---

# ■ Public IP

インターネット公開用

例：

`20.x.x.x`

---

# 🎯 試験ポイント

👉 外部公開にはPublic IP必要

---

# ⚖️ **Static vs Dynamic**

| 種類 | 特徴 |
| --- | --- |
| **Static** | 固定 |
| **Dynamic** | 自動割当 |

---

# 🎯 試験ポイント

👉 Public IPでよく出る

---

# 🔐 **NSG（超重要）**

---

# 🔰 概要

**Network Security Group**

👉 通信制御ルール

---

# 💡 例

| ポート | 用途 |
| --- | --- |
| `22` | SSH |
| `3389` | RDP |
| `80` | HTTP |
| `443` | HTTPS |

---

# 🎯 試験ポイント

👉 接続不可問題で頻出

# 🔰 概要

---

## 🎯 試験ポイント

# 🛠️ **VNet作成（CLI）**

---

## ■ VNet作成

```
az network vnet create \
--resource-group myRG \
--name myVNet \
--address-prefix10.0.0.0/16 \
--subnet-name default \
--subnet-prefix10.0.1.0/24
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--address-prefix` | VNet範囲 |
| `--subnet-name` | サブネット名 |
| `--subnet-prefix` | サブネット範囲 |

---

# 🛠️ **NSG作成**

---

## ■ NSG作成

```
az network nsg create \
--resource-group myRG \
--name myNSG
```

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| VMはVNet不要 | ❌ |
| Public IPなしでインターネット接続 | ❌ |
| NSGは認証管理 | ❌ |
