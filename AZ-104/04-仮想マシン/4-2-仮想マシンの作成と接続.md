# 🖥️ **仮想マシン（VM）の作成と接続**

ここはAZ-104でも超重要です 👍

実務でも最もよく使う分野の1つです。

試験では特に👇が頻出です

- VM作成時の設定
- 接続方法
- ネットワーク構成
- SSH / RDP
- Public IP
- NSG

---

# 🔰 **VM作成時に必要なもの**

Azure VMは単体では動きません。

作成時に複数のリソースが必要です👇

```
VM
 ├ OSディスク
 ├ NIC（ネットワーク）
 ├ VNet
 ├ サブネット
 └ Public IP（必要なら）
```

---

# 🎯 **試験ポイント**

👉 VM作成時にAzureが自動作成するものが問われる

---

# 🧱 **VM作成時の主な設定**

---

# ① ⚙️ **VMサイズ**

## 🔰 概要

性能を決める

例：

- `Standard_B2s`
- `Standard_D2s_v5`

## 🎯 試験ポイント

- サイズ変更可能
- CPU・メモリ量が変わる

---

# ② 💽 **OSイメージ**

## 🔰 概要

インストールするOS

例：

- Ubuntu
- Windows Server

---

## CLI例

```
--image Ubuntu2204
```

---

# ③ 👤 **管理者アカウント**

## Linux

- SSH鍵認証

---

## Windows

- RDP接続

---

## 🎯 試験ポイント

👉 LinuxはSSHが基本

---

# ④ 🌐 **ネットワーク**

VMには基本的に👇が必要

| 項目 | 内容 |
| --- | --- |
| **NIC** | ネットワークカード |
| **VNet** | 仮想ネットワーク |
| **サブネット** | ネットワーク区画 |
| **Public IP** | インターネット接続 |

---

# 🛠️ **VM作成（CLI）**

---

# ■ Linux VM作成

```
az vm create \
--resource-group myRG \
--name myVM \
--image Ubuntu2204 \
--size Standard_B2s \
--admin-username azureuser \
--generate-ssh-keys
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--resource-group` | 配置先 |
| `--name` | VM名 |
| `--image` | OS |
| `--size` | VMサイズ |
| `--admin-username` | 管理者 |
| `--generate-ssh-keys` | SSH鍵生成 |

---

# 🎯 試験ポイント

👉 `--generate-ssh-keys` はLinuxで重要

---

# 🔐 **VMへの接続方法**

---

# ① 🐧 **SSH（Linux）**

## 🔰 概要

Linux接続方法

---

## 接続コマンド

```
ssh azureuser@<PublicIP>
```

---

## 💡 ポイント

- `azureuser` → ユーザー名
- `PublicIP` → VMのIP

---

## 🎯 試験ポイント

👉 Linux = SSH（22番ポート）

---

# ② 🪟 **RDP（Windows）**

## 🔰 概要

Windowsリモート接続

---

## 接続ポート

`3389`

---

## 🎯 試験ポイント

👉 Windows = RDP

---

# 🔥 **NSG（超重要）**

---

# 🔰 NSGとは

**Network Security Group**

👉 通信制御ルール

---

## 💡 例

| ポート | 用途 |
| --- | --- |
| `22` | SSH |
| `3389` | RDP |
| `80` | HTTP |
| `443` | HTTPS |

---

# 🎯 試験ポイント

👉 接続できない原因で頻出

---

# 🛠️ **ポート開放**

---

## SSH許可

```
az vm open-port \
--resource-group myRG \
--name myVM \
--port22
```

---

## RDP許可

```
az vm open-port \
--resource-group myRG \
--name myVM \
--port3389
```

---

# 🔐 **Public IP（重要）**

## 🔰 概要

インターネット公開用IP

---

## 💡 ポイント

- 外部接続時に必要
- 不要なら付与しない

---

## 🎯 試験ポイント

👉 セキュリティ上は最小化

---

# 🔥 **よくあるトラブル（試験頻出）**

---

## ❌ SSH接続できない

原因：

- NSG未許可
- Public IPなし

---

## ❌ RDP接続できない

原因：

- `3389` 閉鎖
- Windows Firewall

---

# 🧠 **Azure Bastion（重要）**

## 🔰 概要

ブラウザ経由で安全接続

---

## 💡 メリット

- Public IP不要
- SSH/RDP可能

---

## 🎯 試験ポイント

👉 セキュア接続で頻出

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| LinuxはRDP接続 | ❌ |
| NSG不要でもSSH可能 | ❌ |
| BastionはVPN必須 | ❌ |

---

# 🎯 **まとめ**

- VM作成には
    - NIC
    - VNet
    - サブネット
    が必要
- Linux接続
    - SSH（22）
- Windows接続
    - RDP（3389）
- 通信制御
    - **NSG**
