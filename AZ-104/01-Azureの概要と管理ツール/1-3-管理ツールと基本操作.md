# 🛠️ **Azureの管理ツール（Portal / CLI / PowerShell）**

Azureは主に3つの方法で操作します👇

- **Azure Portal（GUI）**
- **Azure CLI（コマンド）**
- **Azure PowerShell（コマンド）**

---

# ① 🖥️ **Azure Portal（GUI）**

## 🔰 概要

ブラウザで操作する管理画面

👉 一番わかりやすい

---

## 💡 特徴

- 視覚的に操作できる
- 初心者向け
- 設定内容を確認しやすい

---

## 🎯 試験ポイント

- 「どの画面で設定するか」が問われることあり
- 実際の構成イメージ理解に重要

---

## ⚠️ 注意点

- 手動操作なので**自動化には不向き**

---

# ② 💻 **Azure CLI**

## 🔰 概要

コマンドでAzureを操作するツール

👉 クロスプラットフォーム（Linux / Mac / Windows）

---

## 💡 基本構文（超重要）

```
az <リソース> <操作> <オプション>
```

👉 例

- `az group create`
- `az vm create`

---

## 🛠️ 基本操作

### ■ ログイン

`az login`

---

### ■ サブスクリプション確認

`az account list`

---

### ■ サブスクリプション切替

`az account set --subscription "<サブスクリプション名>"`

---

### ■ リソースグループ作成

```
az group create \
--name myRG \
--location japaneast
```

---

### ■ VM作成

```
az vm create \
--resource-group myRG \
--name myVM \
--image Ubuntu2204 \
--admin-username azureuser \
--generate-ssh-keys
```

---

## 💡 よく使うオプション

| オプション | 意味 |
| --- | --- |
| `--name` | リソース名 |
| `--resource-group` | 対象グループ |
| `--location` | リージョン |
| `--image` | OSイメージ |

---

## 🎯 試験ポイント

- コマンドの「意味」を問われる
- 完全暗記ではなく
👉 **構造理解が重要**

---

# ③ ⚡ **Azure PowerShell**

## 🔰 概要

PowerShell形式でAzureを操作

---

## 💡 CLIとの違い

| 項目 | Azure CLI | PowerShell |
| --- | --- | --- |
| 形式 | `az`コマンド | `Verb-Noun`形式 |
| OS | 全OS | Windows寄り |
| データ形式 | JSON | オブジェクト |

---

## 🛠️ 基本操作

### ■ ログイン

`Connect-AzAccount`

---

### ■ リソースグループ作成

```
New-AzResourceGroup \
-Name"myRG" \
-Location"japaneast"
```

---

## 🎯 試験ポイント

- CLIとの違いを問われることあり
- **Verb-Noun形式**を理解

---

# 🔥 **3つのツールの使い分け（超重要）**

| ツール | 向いている用途 |
| --- | --- |
| **Portal** | 確認・初心者操作 |
| **CLI** | 自動化・スクリプト |
| **PowerShell** | Windows管理・高度操作 |

---

# ⚠️ **試験での引っかけ**

| 問題 | 正解 |
| --- | --- |
| Portalは自動化に向いている | ❌ |
| CLIはWindows専用 | ❌（全OS対応） |
| PowerShellはCLIと同じ | ❌（構文が違う） |

---

# 🧠 **実務での使い分けイメージ**

- 初期構築 → **Portal**
- 繰り返し処理 → **CLI**
- 詳細制御 → **PowerShell**

---

# 🎯 **まとめ**

- Azureは3つの方法で操作
    - **Portal（GUI）**
    - **CLI（コマンド）**
    - **PowerShell（コマンド）**
- CLIは試験でも実務でも重要
- **自動化＝CLI or PowerShell**

---

# 📊 **比較まとめ（Notion対応）**

## ■ 管理ツール比較

| ツール | 特徴 | 向いている用途 |
| --- | --- | --- |
| **Portal** | GUI操作 | 初心者・確認 |
| **CLI** | `az`コマンド | 自動化・スクリプト |
| **PowerShell** | `Verb-Noun`形式 | Windows・詳細操作 |

---

## ■ CLI vs PowerShell

| 項目 | CLI | PowerShell |
| --- | --- | --- |
| コマンド形式 | `az vm create` | `New-AzVM` |
| データ形式 | JSON | オブジェクト |
| 学習難易度 | 低め | やや高い |
