# 🔐 **RBAC（ロールベースアクセス制御）とは**

## 🔰 **概要**

**RBAC（Role-Based Access Control）** とは

👉 **ユーザーに対して操作権限を割り当てる仕組み**

---

## 💡 一言でいうと

👉 **「誰が何できるか」を決める**

---

## 🧠 **イメージ（超重要）**

```
ユーザー（またはグループ）
        ↓
ロール（権限）
        ↓
リソース（対象）
```

---

## 🎯 **試験ポイント（最重要）**

- **RBAC = 認可（Authorization）**
- **Entra ID = 認証（Authentication）**

👉 この違いは必ず出ます

---

# 🧱 **RBACの3要素（超重要）**

## ① **セキュリティプリンシパル**

👉 権限を付与する対象

- **ユーザー**
- **グループ**
- **サービスプリンシパル**

---

## ② **ロール定義（Role）**

👉 できる操作の集合

---

## ③ **スコープ（Scope）**

👉 適用範囲

- サブスクリプション
- リソースグループ
- リソース

---

## 🎯 試験ポイント

👉 この3つの組み合わせで権限が決まる

---

# 🏷️ **代表的なロール（頻出）**

| ロール | 内容 |
| --- | --- |
| **Owner** | すべて可能（権限付与も可） |
| **Contributor** | 作成・変更可能（権限付与不可） |
| **Reader** | 閲覧のみ |

---

## 🎯 試験ポイント（超重要）

👉 **Ownerだけが権限付与できる**

---

# 🧠 **スコープの階層（超重要）**

```
サブスクリプション
 └ リソースグループ
    └ リソース
```

---

## 💡 ポイント

👉 上位で付与すると下位にも適用される

---

## 🎯 試験ポイント

- サブスクリプションで付与 → 全体に適用
- リソース単位で付与 → 個別

---

# 🔥 **よく出る設計（超重要）**

## ❌ NG

- ユーザーに直接付与

---

## ✅ OK

- グループに付与

👉 理由

- 管理しやすい
- 人の増減に強い

---

# 🛠️ **CLI操作（重要）**

---

## ■ ロール割り当て

```
az role assignment create \
--assignee <ユーザーUPN> \
--role"Contributor" \
--scope"/subscriptions/<サブスクリプションID>"
```

---

## 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--assignee` | 対象ユーザー |
| `--role` | ロール |
| `--scope` | 適用範囲 |

---

## ■ 割り当て確認

```
az role assignment list--output table
```

---

# 🔗 **Azure Policyとの違い（超頻出）**

| 項目 | RBAC | Policy |
| --- | --- | --- |
| 目的 | 権限管理 | 制限 |
| 対象 | ユーザー | リソース |
| タイミング | 操作時 | 作成時 |

---

## 💡 超重要まとめ

- **RBAC → 人を制御**
- **Policy → リソースを制御**

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| RBACでリソース制限する | ❌（Policy） |
| Contributorは権限付与できる | ❌ |
| スコープは1種類のみ | ❌ |

---

# 🎯 **まとめ**

- **RBAC = 権限管理（認可）**
- 3要素
    - **セキュリティプリンシパル**
    - **ロール**
    - **スコープ**
- 代表ロール
    - **Owner / Contributor / Reader**
- ベストプラクティス
    - **グループに付与**

---

# 📊 **比較まとめ（Notion対応）**

## ■ RBACの構成

| 要素 | 内容 |
| --- | --- |
| **プリンシパル** | ユーザー・グループ |
| **ロール** | 権限 |
| **スコープ** | 適用範囲 |

---

## ■ ロール比較

| ロール | 権限 |
| --- | --- |
| **Owner** | フルアクセス + 権限付与 |
| **Contributor** | 作成・変更 |
| **Reader** | 閲覧のみ |

---

## ■ RBAC vs Policy

| 項目 | RBAC | Policy |
| --- | --- | --- |
| 目的 | 権限 | 制限 |
| 対象 | ユーザー | リソース |
| タイミング | 操作時 | 作成時 |
