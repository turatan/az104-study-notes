# 🔗 **VNetピアリング（Peering）**

ここはAZ-104ネットワーク分野で超頻出です 👍

特に、

👉 **VNet同士の接続**

👉 **リージョン間通信**

👉 **Gateway Transit**

👉 **VPNとの違い**

がよく出ます。

---

# 🔰 **VNetピアリングとは**

## ■ 概要

**VNet Peering** とは

👉 **2つのVNetを直接接続する機能**

---

## 💡 一言でいうと

👉 **「Azure内VNet同士の高速専用接続」**

---

# 🧠 **イメージ（超重要）**

```
VNet-A
   ↔
VNet-B
```

---

## 🎯 試験ポイント

- Azureバックボーンネットワーク利用
- インターネットを経由しない

---

# 🌍 **なぜ使うのか**

通常はVNet同士は通信できません。

---

## ❌ ピアリングなし

```
VNet-A   ×   VNet-B
```

---

## ✅ ピアリングあり

```
VNet-A   ↔   VNet-B
```

---

# 🎯 試験ポイント

👉 VNet間通信 = ピアリング

---

# ⚡ **特徴（超重要）**

| 特徴 | 内容 |
| --- | --- |
| **高速** | Azureバックボーン |
| **低遅延** | 内部通信 |
| **Private IP通信** | Public IP不要 |
| **高帯域** | 高速転送 |

---

# 🎯 試験ポイント

👉 Public Internetを通らない

---

# 🌐 **ピアリングの種類**

---

# ① 🌍 **リージョン内ピアリング**

同一リージョン

例：

- Japan East ↔ Japan East

---

# ② 🌎 **グローバルピアリング**

別リージョン

例：

- Japan East ↔ West US

---

# 🎯 試験ポイント

👉 リージョン跨ぎ = Global Peering

---

# 🔐 **通信条件（超重要）**

---

# ■ IP範囲重複禁止

❌ 不可

```
VNet-A: 10.0.0.0/16
VNet-B: 10.0.0.0/16
```

---

## ✅ OK

```
VNet-A: 10.0.0.0/16
VNet-B: 10.1.0.0/16
```

---

# 🎯 試験ポイント

👉 重複アドレス不可

---

# 🔥 **Gateway Transit（超重要）**

---

# 🔰 概要

他VNetのVPN Gateway利用

---

## 💡 イメージ

```
オンプレ
   ↓
VPN Gateway
   ↓
VNet-A
   ↓
Peering
   ↓
VNet-B
```

---

## 🎯 試験ポイント

👉 Gateway共有可能

---

# ⚖️ **VNet Peering vs VPN Gateway（超頻出）**

| 項目 | VNet Peering | VPN Gateway |
| --- | --- | --- |
| 接続先 | Azure VNet | オンプレ / Azure |
| 通信 | Azure Backbone | VPN |
| 速度 | 高速 | やや低速 |
| 暗号化 | 基本なし | VPN暗号化 |

---

# 🎯 超重要

👉 Azure内通信なら通常Peering

---

# 🛠️ **CLIでピアリング作成**

---

## ■ ピアリング作成

```
az network vnet peering create \
--resource-group myRG \
--name PeerToVNet2 \
--vnet-name VNet1 \
--remote-vnet VNet2 \
--allow-vnet-access
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--vnet-name` | 接続元 |
| `--remote-vnet` | 接続先 |
| `--allow-vnet-access` | 通信許可 |

---

# 🔄 **双方向設定（重要）**

---

# 🔰 概要

両VNet側に設定必要

---

## 🎯 試験ポイント

👉 片側だけでは不十分

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| ピアリングはインターネット経由 | ❌ |
| 重複IPでも接続可能 | ❌ |
| 片側設定だけでOK | ❌ |

---

# 🎯 **まとめ**

- VNet Peering = VNet間接続
- 特徴
    - 高速
    - 低遅延
    - Private通信
- 超重要
    - **IP重複不可**
    - **Global Peering**
    - **Gateway Transit**

---

# 📊 **比較まとめ（Notion対応）**

## ■ ピアリング種類

| 種類 | 内容 |
| --- | --- |
| **VNet Peering** | 同一リージョン |
| **Global Peering** | 別リージョン |

---

## ■ Peering vs VPN

| 項目 | Peering | VPN |
| --- | --- | --- |
| 通信 | Azure内部 | VPN |
| 速度 | 高速 | 普通 |
| 主用途 | VNet接続 | オンプレ接続 |

---

## ■ 制限事項

| 項目 | 内容 |
| --- | --- |
| **IP重複** | 不可 |
| **双方向設定** | 必須 |
