# 🔰 **コンテナーとは**

## ■ 概要

**コンテナー** とは

👉 **アプリ実行環境をまとめて動かす仕組み**

---

## 💡 一言でいうと

---

```
コンテナー
 ├ アプリ
 ├ ライブラリ
 └ 設定
```

---

## 🎯 試験ポイント

👉 「どこでも同じ動作」が目的

---

# ⚖️ **VMとの違い（超重要）**

| 項目 | VM | コンテナー |
| --- | --- | --- |
| OS | 個別必要 | ホスト共有 |
| 起動 | 遅い | 高速 |
| 容量 | 大きい | 小さい |
| 軽量性 | 低 | 高 |

---

# 🚀 **Azure Container Instances（ACI）**

---

# 🔰 概要

単体コンテナー実行

---

## 💡 特徴

- サーバー管理不要
- 簡単
- 小規模向け

---

# 🎯 試験ポイント

👉 簡易実行 = ACI

---

# 🧠 イメージ

```
ACI
 └ Container
```

---

# ☸️ **Azure Kubernetes Service（AKS）**

---

# 🔰 概要

Kubernetes管理サービス

---

## 💡 一言でいうと

👉 **「大量コンテナー管理サービス」**

---

# 🧠 イメージ

```
AKS
 ├ Container1
 ├ Container2
 └ Container3
```

---

# 🎯 試験ポイント

👉 大規模運用 = AKS

---

# ☸️ **Kubernetesとは**

---

# 🔰 概要

コンテナーオーケストレーション

---

## 💡 できること

- 自動配置
- 自動復旧
- スケーリング

---

# 🎯 試験ポイント

👉 AKS = Kubernetes管理版

---

# 🔥 **ACI vs AKS（超重要）**

| 項目 | ACI | AKS |
| --- | --- | --- |
| 規模 | 小 | 大 |
| 管理 | 簡単 | 高機能 |
| Kubernetes | × | 〇 |
| 用途 | 単体実行 | 本番運用 |

---

# 📦 **Azure Container Registry（ACR）**

---

# 🔰 概要

Dockerイメージ保存場所

---

## 💡 イメージ

```
Docker Image
     ↓
ACR
     ↓
AKS/ACI
```

---

# 🎯 試験ポイント

👉 ACR = Azure版Docker Hub

---

# 🛠️ **CLIでACI作成**

---

## ■ ACI作成

```
az container create \
--resource-group myRG \
--name mycontainer \
--image nginx
```

---

# 🛠️ **AKS作成**

```
az aks create \
--resource-group myRG \
--name myAKS \
--node-count2 \
--generate-ssh-keys
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--node-count` | ノード数 |
| `--image` | コンテナーイメージ |

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| コンテナーはGuest OS必要 | ❌ |
| AKSは単体コンテナー専用 | ❌ |
| ACRは実行環境 | ❌ |
