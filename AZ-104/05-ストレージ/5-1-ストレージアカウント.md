# 💾 **ストレージアカウント（Storage Account）**

ここはAZ-104で超重要です 👍

ストレージ分野の土台であり、

👉 **Blob**

👉 **Files**

👉 **冗長性**

👉 **アクセス制御**

すべての基礎になります。

---

# 🔰 **ストレージアカウントとは**

## ■ 概要

**ストレージアカウント** とは

👉 **Azureストレージサービスを利用するための箱**

---

## 💡 一言でいうと

👉 **「Azureの保存領域の親コンテナ」**

---

# 🧠 **イメージ（超重要）**

```
ストレージアカウント
 ├ Blob Storage
 ├ File Shares
 ├ Queue
 └ Table
```

---

## 🎯 試験ポイント

- ストレージサービスの土台
- 必ず先に作成する

---

# 🧱 **ストレージサービスの種類**

| サービス | 用途 |
| --- | --- |
| **Blob** | ファイル保存 |
| **Files** | SMB共有 |
| **Queue** | メッセージ |
| **Table** | NoSQL |

---

# 🎯 AZ-104で重要なのは？

👉 **Blob と Files**

---

# 🔥 **ストレージアカウント作成時の重要項目**

---

# ① 🌍 **リージョン**

例：

- `japaneast`

---

# ② ⚙️ **パフォーマンス**

| 種類 | 特徴 |
| --- | --- |
| **Standard** | 一般用途 |
| **Premium** | 高性能 |

---

## 🎯 試験ポイント

👉 DBや高IOPS用途 → Premium

---

# ③ 🔁 **冗長性（超重要）**

ここは頻出です。

---

# ■ LRS（Locally Redundant Storage）

👉 同一データセンター内で3重保存

---

# ■ ZRS（Zone-Redundant Storage）

👉 可用性ゾーン間で保存

---

# ■ GRS（Geo-Redundant Storage）

👉 別リージョンにも複製

---

# ■ RA-GRS

👉 GRS + 読み取り可能

---

# 🎯 超重要比較

| 種類 | 複製範囲 |
| --- | --- |
| **LRS** | 同一DC |
| **ZRS** | 同一リージョン別ゾーン |
| **GRS** | 別リージョン |
| **RA-GRS** | GRS + 読み取り |

---

# 🧠 **アクセス層（Access Tier）**

Blob Storageで重要

---

# ■ Hot

👉 頻繁アクセス

---

# ■ Cool

👉 低頻度アクセス

---

# ■ Archive

👉 長期保管

---

## 🎯 試験ポイント

👉 安いほど取り出しが遅い

---

# 🔐 **セキュリティ（重要）**

---

# ■ アクセスキー

ストレージの認証情報

---

# ■ SAS（Shared Access Signature）

一時的アクセス権

---

# ■ Azure AD認証

Entra ID利用

---

## 🎯 試験ポイント

👉 外部共有 → **SAS**

---

# 🛠️ **CLIで作成**

---

## ■ ストレージアカウント作成

```
az storage account create \
--name mystorage123 \
--resource-group myRG \
--location japaneast \
--sku Standard_LRS
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--name` | ストレージ名 |
| `--sku` | 冗長性・性能 |
| `--location` | リージョン |

---

# 🎯 試験ポイント

👉 `Standard_LRS`

- Standard = 性能
- LRS = 冗長性

---

# 🔥 **ストレージアカウント名のルール（頻出）**

| ルール | 内容 |
| --- | --- |
| 小文字のみ | 必須 |
| グローバル一意 | 必須 |
| 3〜24文字 | 制限 |

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| BlobはVM用ディスク | ❌ |
| GRSは同一リージョン複製 | ❌ |
| ストレージ名は重複可能 | ❌ |
