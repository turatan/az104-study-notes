# 🔥 **AzCopy（超重要）**

---

# 🔰 概要

高速データ転送CLIツール

---

# 💡 用途

- 大容量転送
- バックアップ
- Blob同期

---

# 🎯 試験ポイント

👉 「大量データ高速転送」なら AzCopy

---

# 🛠️ **AzCopy基本構文**

```
azcopy copy"<コピー元>""<コピー先>"
```

---

# 📤 **アップロード例**

```
azcopy copy"./data" \
"https://mystorage123.blob.core.windows.net/container?<SAS>" \
--recursive
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `copy` | コピー |
| `--recursive` | 再帰コピー |

---

# 📥 **ダウンロード例**

```
azcopy copy \
"https://mystorage123.blob.core.windows.net/container?<SAS>" \
"./download" \
--recursive
```

---

# 🔄 **同期（sync）**

---

# 🔰 概要

差分同期

---

## ■ 例

```
azcopy sync \
"./data" \
"https://mystorage123.blob.core.windows.net/container?<SAS>"
```

---

# 🎯 試験ポイント

👉 `copy` と `sync` の違い

---

# ⚖️ **copy vs sync（重要）**

| コマンド | 特徴 |
| --- | --- |
| `copy` | 全コピー |
| `sync` | 差分同期 |
