---

# 🔰 **Azure Filesとは**

## ■ 概要

**Azure Files** とは

👉 **Azure上で利用できるファイル共有サービス**

---

## 💡 一言でいうと

👉 **「クラウド上の共有フォルダ」**

---

# 🧠 **イメージ（超重要）**

```
ストレージアカウント
 └ File Share
     └ ファイル
```

---

## 🎯 試験ポイント

- SMBベースの共有
- Windows共有フォルダに近い

---

# 🔥 **Blobとの違い（超頻出）**

| 項目 | Azure Files | Blob Storage |
| --- | --- | --- |
| 形式 | ファイル共有 | オブジェクト |
| プロトコル | SMB / NFS | HTTPS |
| 用途 | 共有フォルダ | ファイル保存 |
| ドライブマウント | 可能 | 不可 |

---

## 🎯 超重要

👉 「共有フォルダ」なら **Azure Files**

---

# 🧱 **Azure Filesの特徴**

---

# ① 📂 **SMB共有**

## 🔰 SMBとは

Windowsファイル共有プロトコル

---

## 💡 できること

- エクスプローラーから接続
- ネットワークドライブ化

---

## 🎯 試験ポイント

👉 SMB = Azure Files

---

# ② 🐧 **NFS対応**

Linux向け

---

# ③ 🌐 **クラウド共有**

オンプレ・Azure VM両方から利用可能

---

# ④ 🔄 **Azure File Sync（超重要）**

---

# 🔰 概要

オンプレファイルサーバーと同期

---

## 💡 イメージ

```
オンプレファイルサーバー
        ↕
Azure Files
```

---

## 🎯 試験ポイント

👉 File Sync = キャッシュ利用可能

---

# 🛠️ **File Share作成（CLI）**

---

## ■ ファイル共有作成

```
az storage share create \
--account-name mystorage123 \
--name fileshare01
```

---

# 💡 オプション解説

| オプション | 内容 |
| --- | --- |
| `--account-name` | ストレージアカウント |
| `--name` | 共有名 |

---

# 🖥️ **Windowsから接続**

---

## ■ ドライブマウント例

```
net use Z: \\mystorage123.file.core.windows.net\fileshare01
```

---

## 💡 ポイント

- `Z:` → ドライブレター
- `\\` → SMB共有パス

---

# 🎯 試験ポイント

👉 Azure Filesはマウント可能

---

# 🔐 **認証方式（重要）**

| 方法 | 内容 |
| --- | --- |
| **アクセスキー** | ストレージ認証 |
| **Entra ID** | Azure認証 |
| **AD DS** | オンプレAD |

---

# 🎯 試験ポイント

👉 Entra IDベース認証が増えている

---

# 💾 **冗長性**

Blob同様👇を選択可能

- **LRS**
- **ZRS**
- **GRS**

---

# 🎯 試験ポイント

👉 冗長性はストレージアカウント単位

---

# 🔥 **Azure File Sync（超重要）**

---

# 🔰 なぜ使う？

通常のオンプレ共有フォルダ👇

❌ 容量不足

❌ バックアップ大変

---

Azure File Sync👇

✅ クラウド同期

✅ キャッシュ保持

✅ DR対策

---

# 🧠 **クラウド階層化（Cloud Tiering）**

## 🔰 概要

よく使うファイルだけローカル保存

---

## 💡 メリット

- ディスク節約
- 高速アクセス

---

## 🎯 試験ポイント

👉 File Sync + Tiering は頻出

---

# ⚠️ **試験の引っかけ**

| 問題 | 正解 |
| --- | --- |
| Azure Filesはオブジェクトストレージ | ❌ |
| BlobはSMB共有可能 | ❌ |
| File Syncは同期できない | ❌ |

---

# 🎯 **まとめ**

- Azure Files = クラウド共有フォルダ
- SMB / NFS対応
- Windows共有に近い
- 超重要
    - **Blobとの違い**
    - **Azure File Sync**
    - **Cloud Tiering**

---

# 📊 **比較まとめ（Notion対応）**

## ■ Azure Files vs Blob

| 項目 | Azure Files | Blob |
| --- | --- | --- |

| SMB | 〇 | × |
